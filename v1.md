MORNING ROUTINE:
  1. Open browser → Streamlit app loads
  2. Click "Sync Inbox" → Fetches last 50 emails
  3. Auto-summarizes each → Shows digest view
  4. Click any email → Full summary with key points

SEARCH:
  1. Type "show me receipts from Amazon"
  2. Embedding model converts query to vector
  3. ChromaDB finds 12 matching emails
  4. Shows subject + date + summary for each

CHAT:
  1. Ask "What did Sarah say about the budget?"
  2. System finds Sarah's emails (ChromaDB + metadata filter)
  3. Feeds top 3 to chat model with prompt
  4. Answers with citations: "Sarah said X on June 10th"

ORGANIZE:
  1. Click "Group by Topic"
  2. Embedding model clusters all inbox vectors
  3. Shows: "Shopping (8), Work (23), Finance (5), News (14)"
  4. Click any group to see those emails
